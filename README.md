# MIRAI - Sales & Interaction SLM
Le pilier "Rapide". L'interface de vente directe et rapide.

## Rôle
- Compréhension de l'intention client.
- Tunnel de vente (conversion).
- Orchestration : appelle ATLAS pour le stock.

## Installation & Usage
```bash
uv pip install -r requirements.txt
# Lancer l'API Gateway pour le Frontend
uv run api.py
```
