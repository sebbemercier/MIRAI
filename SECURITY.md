# Security Policy

## Supported Versions

Only the latest version of MIRAI is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability within MIRAI, please send an e-mail to security@openslm.org. All security vulnerabilities will be promptly addressed.

Please include the following information in your report:
- Type of issue (e.g., Prompt injection, session hijacking, etc.)
- Location of the issue (file, function, etc.)
- Potential impact
- Steps to reproduce the issue

We ask you to not disclose the vulnerability publicly until we have had a chance to address it.
