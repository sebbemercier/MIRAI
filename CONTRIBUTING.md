# Contributing to MIRAI

Thank you for your interest in contributing to MIRAI! MIRAI is the sales and interaction agent of the OpenSLM ecosystem.

## Specific Focus
MIRAI is responsible for understanding customer intent and driving sales conversion. Contributions are welcome in the following areas:
- Intent classification models for e-commerce.
- Natural language generation for sales dialogues.
- Integration with messaging platforms (WhatsApp, Slack, etc.).
- Multi-language support (French, Dutch, English).

## How to Contribute
1. **Fork the Repository**: Create your own fork of the repository.
2. **Clone the Fork**: Clone your fork to your local machine.
3. **Create a Branch**: Create a new branch for your feature or bugfix.
4. **Make Changes**: Implement your changes, following the project's coding standards.
5. **Run Tests**: Ensure that your changes do not break existing functionality.
6. **Submit a Pull Request**: Submit a pull request to the main repository.

## Coding Standards
- Use `uv` for dependency management.
- Follow PEP 8 for Python code style.
- Include unit tests for new features.
- Document your changes clearly.
